# Media assets

Screenshots and videos (GIFs) referenced from the top-level README live here.
Drop in files with the exact names below and they'll show up in the README automatically (broken-image until added).

Prefer GIF or mp4. Use **GIF** if you want it to auto-play in the README; mp4 works too if you upload it to an Issue/Release and reference the URL.

| Filename | Type | What to capture |
|---|---|---|
| `hero.png` | image | The top-down office overview. The "face" shot showing the orchestrator (main session) and employees (agents). |
| `hooks-timeline.gif` | video | The moment a hook fires, animated — making clear when and which hook ran over time. |
| `hooks-manager.png` | image | The Hooks screen: the effective hook setup for a session (including plugin-provided hooks). |
| `multi-session.gif` | video | Multiple Claude sessions shown on one screen, all progressing at the same time. |
| `agents-skills.png` | image | The Agents / Skills screen: which agents and skills are defined / active. |
| `terminal-focus.gif` | video | Clicking an employee/session brings the terminal that launched it to the front. |
| `metrics.png` | image | The management dashboard (utilization, call count, failure rate, tokens). |

## Guidelines
- Images: PNG around 1200–1600px wide.
- GIFs: 5–15s, aim for under 5MB (large files slow down README rendering). For bigger clips, use mp4 hosted via a Release/Issue.
