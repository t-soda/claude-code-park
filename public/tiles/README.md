# Tile PNGs (floor / wall)

Place the 1-cell tiles for the floor and walls here. See `public/objects/README.md` for details.

- `floor.png` … floor (128×64 diamond)
- `wall-left.png` … wall, lower-left facing (back-left edge, 128 wide × free height)
- `wall-right.png` … wall, lower-right facing (back-right edge, 128 wide × free height)

If missing, falls back to code drawing (checkerboard + 3D walls).
